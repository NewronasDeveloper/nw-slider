NW SLIDER

Nw slider is a responsive carousel very simple yet enough to make an elegant slider of images.

It works with a javascript library with only 2KB in size and does not require jQuery.

Transitions fall on css styles, yet the nw slider css file weighs just under 4KB.

The propositios of nw slider are: light and simple.

Put a nice carousel on your website with only 6KB in size.

Do I know to use nw slider?

  1. Implements the code of nw slider, css and js inside the head tag.
    (change the path if necessary to nw-slider.min.css and to nw-slider.min.js)
    (nw-slider.min.js can be loaded inside the head or inside the body)
    <head>
    //your head code
    <link rel="stylesheet" href="assets/css/nw-slider.min.css" type="text/css">
    <script src="assets/js/nw-slider.min.js">
    </head>
  2. Create a div with the css properties you want (it can be dynamic) nw slider will automatically adjust its width and height.
    <div style="width:450px">
    </div>
  3. Inside the previous div create a div with the id you want (this id is mandatory) and make a list of images inside this using the label ul.
    <div style="width:450px">
    <div id="my-id">
    <ul>
    <li><img src="image_1.png" /></li>
    <li><img src="image_2.png" /></li>
    <li><img src="image_3.png" /></li>
    </ul>
    </div>
    </div>
  4. Now I just need the javascript.
    <script>
    NW_SLIDER.add({
    id: 'my-id'
    });
    </script>
    You already have your slider running!
    NW_SLIDER.add is the javascript method for the slider to work. The only mandatory parameter is the id of the div containing the list of images. Other possible parameters are explained later.

  5. To add another slider to the same page, follow steps 2, 3 and 4. All you have to do is assign a different id.

Configuration

Available transitions

  • linear-left
  • linear-right
  • linear-top
  • linear-bottom
  • top-center
  • none
  • corner
  • opacity

How to assign a transition?

Add the transition property in the NW method SLIDER.add and assign the transition name as the value.

<script>
NW_SLIDER.add({
id: 'my-id',
transition: 'opacity'
});
</script>

Keep in mind
If the transition property is not defined, it acts linear-left by default.

How to set the speed?

Add the speed property in the NW method SLIDER.add and assign number of milliseconds as the value.

<script>
NW_SLIDER.add({
id: 'nw-slider',
transition: 'top-center',
speed: 3000
});
</script>

Keep in mind
If the speed property is not defined, it acts 5000 milliseconds by default.
At least must allocate 2000 milliseconds, except the none transition that has no minimun limit and the opacity transition that at least 6000 milliseconds.

How to add buttons to pass the images?

Add the buttons property in the NW method SLIDER.add and assign true as the value.

<script>
NW_SLIDER.add({
id: 'nw-slider',
transition: 'top-center',
speed: 3000,
buttons: true
});
</script>

Keep in mind
If the buttons property is not assigned, the buttons will not appear by default.

How to add text, links ... to the slider?

Inside the <li> tag, add the <span> </span> tags and type between them.

<div style="width:450px">
<div id="my-id">
<ul>
<li>
<img src="image_1.png" />
<span>some text here for image 1</span>
</li>
<li>
<img src="image_2.png" />
<span>some text here for image 2</span>
</li>
<li>
<img src="image_3.png" />
<span>some text here for image 3</span>
</li>
</ul>
</div>
</div>
